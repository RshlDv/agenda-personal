# Tu Agenda Personal
Una aplicación web diseñada para la gestión de eventos y tareas diarias que permite a los usuarios organizar su rutina mediante la creación de eventos.

---
### Tecnologías:

* **Backend:** Java, Spring Boot
* **Frontend:** HTML, CSS, JavaScript
* **Motor de Plantillas:** Thymeleaf
* **Estilos:** Tailwind CSS
* **Iconos:** FontAwesome
* **Otros:** Microsoft Edge Tools, Webhint
---

### Requisitos Previos
* **Java JDK** (versión 17 o superior recomendada)
* **Maven** instalado (o puedes usar el wrapper `./mvnw` incluido en el proyecto)

### Pasos para la instalación
1.  **Clonar el repositorio:**

2.  **Construir el proyecto:**
    ```bash
    ./mvnw clean install
    ```
3.  **Ejecutar la aplicación:**
    ```bash
    ./mvnw spring-boot:run
    ```
4.  **Acceder a la aplicación:**
    Desde tu navegador de preferencia ingresando esta url:
    ```text
    http://localhost:8080
    ```
    Puedes modificar el puerto después junto a otras configuraciones importantes en el archivo `application.properties`.

    Y listo, organicemos nuestros días :D
